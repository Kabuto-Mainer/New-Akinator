# GKN - GraphicKabutoNovel

GKN - это основа для создания виртуальных новелл. В нем можно прописать сюжет и необходимые текстуры для интерфейса и просто запускать программу.
Настройка выполняется в файлах конфигурации во директории `Configs` и `Data`.

В `Configs` вы можете настраивать показываемые на экране объекты, однако делать это надо аккуратно, т.к. на данный момент это не настроено для редактирования.

Во вкладке `Data` вы можете настраивать текстуры и сюжеты новеллы.
Текстуры хранятся в папке `Img`
